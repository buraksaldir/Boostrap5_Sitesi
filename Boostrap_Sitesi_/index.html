Navs and tabs (Gezinmeler ve sekmeler)

Bootstrap'in içerdiği gezinme bileşenlerinin nasıl kullanılacağına ilişkin belgeler ve örnekler.

Base nav (ana navigasyon)

Bootstrap'te bulunan gezinme, temel .nav sınıfından etkin ve devre dışı durumlarına kadar genel işaretlemeyi ve stilleri paylaşır. Her stil arasında geçiş yapmak için değiştirici sınıfları değiştirin. Temel .nav bileşeni, flexbox ile oluşturulmuştur ve her tür gezinme bileşenini oluşturmak için güçlü bir temel sağlar. Bazı stil geçersiz kılmaları (listelerle çalışmak için), daha büyük isabet alanları için bazı bağlantı dolguları ve devre dışı bırakılan temel stiller içerir.

Sınıflar baştan sona kullanılır, böylece işaretlemeniz son derece esnek olabilir. Yukarıdaki gibi < ul>'leri kullanın, öğelerinizin sırası önemliyse < ol>'yi kullanın veya < nav> öğesiyle kendinizinkini yuvarlayın. .nav, display: flex'i kullandığından, gezinme bağlantıları, gezinme öğeleriyle aynı şekilde davranır, ancak ekstra işaretleme yoktur.

Regarding accessibility (Erişilebilirlikle ilgili)

Gezinme çubuğu sağlamak için navs kullanıyorsanız, bir role="navigation" en mantıksal ana kapsayıcısına < ul> veya sarın < nav> tüm navigasyonun etrafındaki öğe. Rolü eklemeyin < ul> çünkü bu, yardımcı teknolojiler tarafından gerçek bir liste olarak duyurulmasını engelleyecektir.

Gezinme çubuklarına, görsel olarak .nav-tabs sınıfıyla sekmeler şeklinde stil verilmiş olsa bile, role="tablist" , role="tab" veya role="tabpanel" nitelikleri verilmemesi gerektiğini unutmayın. Bunlar yalnızca WAI ARIA Yazma Uygulamalarında açıklandığı gibi dinamik sekmeli arayüzler için uygundur. Örnek için bu bölümdeki dinamik sekmeli arayüzlere yönelik JavaScript davranışına bakın. JavaScript'imiz seçilen durumu etkin sekmeye aria-selected='true' ekleyerek işlediğinden dinamik sekmeli arayüzlerde aria-current özelliği gerekli değildir.

JavaScript behavior (JavaScript davranışı)

Gezinme sekmelerimizi ve haplarımızı yerel içeriğin sekmeli bölmeleri oluşturacak şekilde genişletmek için sekme JavaScript eklentisini kullanın (tek tek veya derlenmiş bootstrap.js dosyası aracılığıyla ekleyin). WAI ARIA Yazma Uygulamalarında açıklandığı gibi dinamik sekmeli arayüzler, yapılarını, işlevlerini ve mevcut durumlarını kullanıcılara iletmek için role = "tablist" , role = "tab" , role = "tabpanel" ve ek aria- niteliklerini gerektirir. yardımcı teknolojiler (ekran okuyucular gibi). En iyi uygulama olarak, sekmeler için < button> öğelerinin kullanılmasını öneririz; çünkü bunlar yeni bir sayfaya veya konuma giden bağlantılar yerine dinamik bir değişikliği tetikleyen kontrollerdir. Dinamik sekmeli arayüzlerin açılır menüler içermemesi gerektiğini unutmayın; çünkü bu, hem kullanılabilirlik hem de erişilebilirlik sorunlarına neden olur. Kullanılabilirlik açısından bakıldığında, o anda görüntülenen sekmenin tetikleyici öğesinin hemen görünür olmaması (kapalı açılır menüde olduğu için) kafa karışıklığına neden olabilir. Erişilebilirlik açısından bakıldığında, şu anda bu tür bir yapıyı standart bir WAI ARIA modeliyle eşleştirmenin mantıklı bir yolu yoktur, bu da bunun yardımcı teknoloji kullanıcıları için kolayca anlaşılır hale getirilemeyeceği anlamına gelir.

JavaScript sekmesi, içeriğin görünürlüğünü ve stilini kontrol etmek için sınıfları değiştirir. Bunu sekmeler, haplar ve diğer .nav destekli navigasyonlarla kullanabilirsiniz.

Via JavaScript (JavaScript aracılığıyla)

Sekilebilir sekmeleri JavaScript aracılığıyla etkinleştirin (her sekmenin ayrı ayrı etkinleştirilmesi gerekir): 


        var triggerTabList = [].slice.call(document.querySelectorAll('#myTab a'))
        triggerTabList.forEach(function (triggerEl) {
          var tabTrigger = new bootstrap.Tab(triggerEl)
        
          triggerEl.addEventListener('click', function (event) {
            event.preventDefault()
            tabTrigger.show()
          })
        })

Tek tek sekmeleri birkaç şekilde etkinleştirebilirsiniz:


        var triggerEl = document.querySelector('#myTab a[href="#profile"]')
        bootstrap.Tab.getInstance(triggerEl).show() // Select tab by name
        
        var triggerFirstTabEl = document.querySelector('#myTab li:first-child a')
        bootstrap.Tab.getInstance(triggerFirstTabEl).show() // Select first tab

Show(göstermek)

Verilen sekmeyi seçer ve ilişkili bölmesini gösterir. Daha önce seçilen diğer sekmelerin seçimi kaldırılır ve ilişkili bölmesi gizlenir. Sekme bölmesi gerçekten gösterilmeden (yani, gösterilen.bs.tab olayı gerçekleşmeden önce) arayana geri döner.


 var someTabTriggerEl = document.querySelector('#someTabTrigger')
 var tab = new bootstrap.Tab(someTabTriggerEl)

 tab.show()

Dispose(elden çıkarmak)

Bir öğenin sekmesini yok eder.

getInstance

Bir DOM öğesiyle ilişkili sekme örneğini almanızı sağlayan statik yöntem


        var triggerEl = document.querySelector('#trigger')
        var tab = bootstrap.Tab.getInstance(triggerEl) // Returns a Bootstrap tab instance

getOrCreateInstance

Bir DOM öğesiyle ilişkili sekme örneğini almanıza veya başlatılmamış olması durumunda yeni bir tane oluşturmanıza olanak tanıyan statik yöntem


            var triggerEl = document.querySelector('#trigger')
            var tab = bootstrap.Tab.getOrCreateInstance(triggerEl) // Returns a Bootstrap tab instance
